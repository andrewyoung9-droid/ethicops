Products

Performance tweaks most people skip

Kame-Ha applies native Windows policies that are already on your PC—just rarely used together. Examples include trimming background tasks via power plans tuned for IO-heavy apps, disabling redundant scheduled tasks, and pausing Search indexing while gaming so disk stays focused on your workload.

Network & services hardening

We reduce surface area by disabling legacy protocols (SMBv1), closing unused inbound rules, and setting service start types to "Demand" for components that don’t need to run 24/7. You can snapshot, apply, and roll back in one click.

Privacy without telemetry

Profiles turn off ad ID, background app permissions, and install-time data sharing. Nothing phones home—no spyware, no analytics SDKs. You keep control with a readable change log.

Safe by design

Only signed binaries; all changes use documented Windows APIs, group policy templates, and PowerShell cmdlets. If a tweak isn’t reversible or violates best practices, it doesn’t ship.

Get-AuthenticodeSignature .\Kame-Ha-setup-x64.exe | Format-List
