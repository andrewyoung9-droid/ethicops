Frequently Asked Questions

Quick answers. If you still need help, write support@ethicops.gt.tc.

Will this break my PC? Can I revert?

Kame-Ha applies documented Windows policies, service start-type changes, and scheduler/power-plan tweaks. A rollback option restores previous settings. When in doubt, create a system restore point first.

Does the app collect telemetry?

No. There is no analytics SDK or tracking. Downloads are signed and can be verified with Authenticode, SHA-256, and PGP.

What’s the difference between Standard and Pro?

  • Standard: one-click performance/privacy profiles; service trimming; legacy protocol hardening.
  • Pro: everything in Standard plus scheduled profiles, extra hardening presets, and rollback checkpoints.

Refund policy?

30-day no-questions refund. See Refunds for steps.

How do I verify the download?

Get-FileHash .\kameha-setup-x64.exe -Algorithm SHA256
gpg --import ethicops_pubkey.asc
gpg --verify kameha-setup-x64.exe.asc kameha-setup-x64.exe

Expected fingerprint: E10C E5D7 FBC3 9023 6E74 D6D0 112F C8AF 5D65 590E

Hardware / OS requirements?

Windows 10/11 x64; ~50 MB disk; admin rights for certain policies.
